triopenisini

Проект для обучения командной работе с Git (Штребух, Пидиди и Баребух)
Быстрый старт

    1.Установите все необходимые расширения для php вместе с composer. Я хз как их устанавливать на винде сами разберетесь

    2.Для запуска локального сервера выполните в терминале (находясь в корне проекта) команду php artisan serve

Структура проекта

Создайте файл .env в корне проекта по шаблону .env.example
Основная директория - ./app/

./app/Http/Controllers/ QuestionController.php — тут теперь вся логика: вывод списка вопросов, сохранение новых постов и их удаление.

./app/Models/ User.php и Question.php — настроены связи hasMany и belongsTo. Теперь вопрос «знает» своего автора.

./resources/views/ questions/index.blade.php — главная страница. Добавлена форма создания и лента с кнопками удаления.

./routes/web.php Прописаны маршруты для GET (просмотр), POST (создание) и DELETE (удаление).
Что мы сделали сегодня (обязательно к прочтению):

    Связи БД: Теперь нельзя просто создать вопрос без user_id. В контроллере пока стоит заглушка на первого юзера.

    Безопасность: В формах обязательно юзаем @csrf, иначе Ларавел выплюнет ошибку 419, тупо не читает без нее вьюшку.

    Удаление: Для удаления используем скрытый метод @method('DELETE') внутри формы кнопки.
