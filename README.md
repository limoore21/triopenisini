# triopenisini
Проект для обучения командной работе с Git (Штребух, Пидиди и Баребух)

## Быстрый старт
1. Установите все необходимые расширения для php вместе с composer. Я хз как их устанавливать на винде
    сами разберетесь
2. Для запуска локального сервера выполните в терминале (находясь в корне проекта) 
команду `php artisan serve` 

## Структура проекта
Создайте файл `.env` в корне проекта по шаблону `.env.example`
### Основная директория - `./app/` 
`./app/Http/Controllers/` Обработка запросов пользователя

`./app/Models/` Хранит модель приложения для работы с бд / данными

`./resources/views/` Здесь прописываем верстку в формате **.blade.php**

`./resources/css/` css файлы

`./routes/web.php` Главный контроллер
### Нюансы
1. После изменения стилей через **tailwild-css** примените в терминале команду `npm run build`
